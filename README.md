# Flashcards

Proyecto para voltear tarjetas, donde se agrega un efecto de transicion con la respuesta.

Se crea un un contenedor por cada tarjeta, luego cada tarjeta en particular, y finalmente el frente y el dorso correspondiente a cada una de las tarjetas.

Se usa un grid layout para crear una cuadricula de 3x3, se anima a cada una de las tarjetas en particular.
