# Flashcards

Proyecto que puede servir para estudiar las capitales del mundo que consiste en voltear las tarjetas para conocer la respuesta, para ello se agrega un efecto de transicion que al pasar el mouse sobre la tarjeta esta muestra la respuesta.

Primero se crea un contenedor general y luego otro por cada tarjeta en particular, y finalmente la propia tarjeta y otro tanto para su frente como para el dorso correspondiente.

Se usa un grid layout para crear una cuadricula de 3x3 y luego se anima a cada una de las tarjetas en particular. 

Finalmente se aplican los estilos con los efectos para cada una de las cuadriculas.
